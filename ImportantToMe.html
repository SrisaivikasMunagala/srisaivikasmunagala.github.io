ISP: Burger Run

The objective of the game is to move around the board eating as many burgers as you can while a shy guy is chasing you.

Motivation

We wanted to create a game based on the game Snake, but we wanted to add a twist. Instead of a Snake collecting apples, Burger Run is Burger man collecting burgers while being chased by enemy, Shy Guy.
